# 💖 3D Anniversary Interactive Album

一个基于 WebGL 和 Three.js 开发的沉浸式 3D 纪念相册。通过引入真实的物理光影与前沿的手势追踪技术，为回忆赋予全新的交互体验。

## ✨ 核心特性

* 🤍 **心形阵列排布**：通过定制的几何逻辑，将 34 张精选照片在 3D 空间中完美构建为心形阵列。
* ✋ **手势追踪交互**：深度集成 MediaPipe 手势识别，实现真实手部动作的隔空操作。
* 💥 **动态物理特效**：支持手势触发的震撼“爆炸”与“收缩”粒子级动态效果。
* 🎨 **极致视觉渲染**：采用 MeshStandardMaterial 材质结合玫瑰金 (Rose Gold) 纹理，打造拟真光影。
* 🖼️ **画廊漫游模式**：支持平滑切换至画廊视角，沉浸式浏览每一帧回忆。

## 🛠️ 技术栈

* **图形渲染**：Three.js / WebGL
* **计算机视觉**：MediaPipe (Hand Tracking)
* **前端基础**：HTML5 / CSS3 / JavaScript (ES6+)

## 🚀 快速启动

由于项目包含加载本地纹理和图片资源的操作，建议通过本地服务器运行，以避免浏览器的跨域限制 (CORS)。

1. 将仓库克隆到本地计算机。
2. 在终端中进入项目根目录。
3. 使用 VS Code 打开项目文件夹，并在扩展商店安装 Live Server 插件。
4. 点击 VS Code 底部状态栏的 Go Live 按钮启动本地服务。
5. 浏览器将自动打开并渲染 3D 相册。
